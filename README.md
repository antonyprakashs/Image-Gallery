Description
This webpage features a curated collection of 8 famous mountain peaks. The layout is designed to be elegant and user-friendly, utilizing a centered horizontal rule and a fluid grid system.

Key Features
Pure Flexbox Layout: Achieves a responsive 4-column grid on desktop that automatically adapts to smaller screens without the need for complex media queries.

Smooth Hover Effects: Utilizes CSS transform: scale() and transition properties to create an immersive zoom effect on the peaks.

Consistent Aspect Ratio: Employs object-fit: cover to ensure all images maintain a uniform look regardless of their original dimensions.

Modern Aesthetics: Features a centered horizontal line and optimized spacing for a professional photography portfolio feel.

How to View
Clone the repository to your local machine.

Locate the index.html file in the project folder.

Open index.html in any modern web browser (Chrome, Firefox, Edge, or Safari).

Resize your browser window to see the Flexbox layout wrap the images automatically.
